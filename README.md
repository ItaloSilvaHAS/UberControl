## Sobre o Projeto

O **UberControl** nasceu da necessidade real de motoristas de aplicativo terem controle sobre seus ganhos e gastos. Sei como e dificil gerenciar o dinheiro quando voce trabalha por conta propria - entre combustivel, manutencao, alimentacao e tantos outros custos, e facil perder a nocao do quanto realmente esta ganhando.

Este app foi criado para resolver esse problema de forma simples e direta.

## Funcionalidades

### Dashboard Inteligente
- **Resumo diario** - Veja seus ganhos, gastos e liquido do dia de forma clara
- **Meta diaria** - Defina quanto quer ganhar por dia e acompanhe seu progresso
- **Resumo do trabalho** - Horas trabalhadas, ganho por hora, km rodados
- **Estatisticas por periodo** - Semana, mes ou todo o historico

### Registro de Ganhos
- Valor recebido
- Aplicativo utilizado (Uber, 99, InDriver, Uber Eats, etc.)
- Data e horario
- Quilometragem rodada
- Horas trabalhadas
- Observacoes

### Registro de Gastos
- Categorias especificas para motoristas:
  - Combustivel (Gasolina/Etanol)
  - Manutencao
  - Lavagem
  - Estacionamento
  - Pedagio
  - Alimentacao
  - Taxa do App
  - Seguro
  - Multa

### Relatorios Visuais
- **Ganhos por Aplicativo** - Descubra qual app te da mais dinheiro
- **Gastos por Categoria** - Saiba para onde seu dinheiro esta indo
- **Evolucao Semanal** - Acompanhe seu progresso ao longo do tempo
- **Desempenho por Dia** - Descubra quais dias da semana sao mais lucrativos

### Outras Funcionalidades
- Historico completo com filtros
- Exportacao e importacao de dados (backup)
- Interface 100% responsiva (otimizada para celular)
- Funciona offline (dados salvos no navegador)
- Modo escuro elegante

## Instalacao

### Opcao 1: Usar Online
Basta acessar o link do deploy e comecar a usar!

### Opcao 2: Rodar Localmente

```bash
# Clone o repositorio
git clone https://github.com/seu-usuario/ubercontrol.git

# Entre na pasta
cd ubercontrol

# Rode o servidor (Python 3)
python server.py

# Ou use qualquer servidor HTTP
npx serve .
```

Acesse `http://localhost:5000` no navegador.

## Como Usar

### 1. Configure sua Meta
Clique em "Editar" na secao de Meta Diaria e defina quanto quer ganhar liquido por dia.

### 2. Registre seus Ganhos
Clique no botao **+** (verde) para adicionar um ganho. Preencha:
- Valor recebido
- Qual app voce usou
- Quantos km rodou
- Quantas horas trabalhou

### 3. Registre seus Gastos
Clique no botao **-** (vermelho) para adicionar um gasto. Escolha a categoria e o valor.

### 4. Acompanhe seus Resultados
- Use o Dashboard para ver seu progresso diario
- Acesse Relatorios para graficos detalhados
- Veja o Historico para revisar todas as transacoes

### 5. Faca Backup
Acesse as configuracoes (icone de engrenagem) e exporte seus dados regularmente!

## Design

O design foi pensado especialmente para motoristas que usam o app enquanto trabalham:
- **Modo escuro** - Reduz cansaco visual durante a noite
- **Botoes grandes** - Faceis de clicar enquanto esta no carro
- **Informacoes claras** - Tudo visivel de relance
- **Mobile-first** - Otimizado para celular
